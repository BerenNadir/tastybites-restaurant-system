# TastyBites Restaurant Management System

TastyBites is a desktop restaurant management system developed in Java using Swing.  
The application allows restaurant staff to manage orders, menu items, billing and kitchen operations through a role-based system.

## Features

- User authentication system
- Role-based access (Manager, Waiter, Chef, Customer)
- Order management
- Menu management
- Billing system
- Kitchen order tracking

## Technologies

- Java
- Java Swing
- Maven
- File-based data storage (Java I/O)

## Project Structure

The application follows object-oriented programming principles with multiple classes responsible for different parts of the system such as authentication, menu management and order processing.

## How to Run

1. Clone the repository
2. Open the project in an IDE (IntelliJ IDEA or VS Code)
3. Build the project using Maven
4. Run the main application file
